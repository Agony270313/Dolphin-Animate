## Detaylı Açıklama
Dolphin Animate is a free, open-source 2D animation application built for frame-by-frame animation.

### Features
- **Onion Skinning** - See previous and next frames while drawing
- **Pressure-Sensitive Brush** - Works with drawing tablets
- **Fill Tool** - Flood fill with anti-aliasing support
- **Eraser** - Clean stroke splitting and fill erasing
- **Layers & Groups** - Organize your animation
- **Auto-Merge** - Strokes merge as you draw
- **Frame Navigation** - Timeline, F5, arrow keys
- **Free Transform** - Move, scale, rotate, skew selected objects
- **Project Management** - Save and load animation projects
- **Keyboard Shortcuts** - Fast workflow with customizable shortcuts

### Tech Stack
- Built with Electron + HTML5 Canvas
- Pure JavaScript, no external dependencies
- Cross-platform support planned (Windows currently)
