# Dolphin Animate v2.0.8

### What's New
* **AI Generate (Beta) Menu is now optional:** You can enable or disable the AI features from the Settings panel.
* **Removed Legacy Models:** Removed Gemini 3.0 Flash to ensure the highest quality results using the latest models.
* **Auto Update Support:** Added full support for automatic updates via GitHub Releases. The app will now notify you when a new version is available!
* **Bug Fixes:** Resolved an issue where the AI modal would freeze after generating an animation.
